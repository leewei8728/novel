# 科幻修真小说创作项目

## 项目概述

本项目是一部200万字的科幻修真融合类型长篇小说的创作资料库，模仿《吞噬星空》风格，但在力量体系、宇宙架构、主角设定、核心主题和叙事风格上进行了全面创新。

## 核心创新点

- **力量体系**：修真+科技双轨并行，最终殊途同归
- **宇宙架构**：三界九天的多维宇宙结构
- **主角设定**：因果之瞳+混沌体，独特的金手指组合
- **核心主题**：探讨科技与修真的本质、永生的意义、力量与责任

## 基础设定

### 主角：陆辰

- **金手指**：因果之瞳（能看到事物之间的因果线）
- **体质**：混沌体（可修炼多种属性，修科融合）
- **性格**：隐忍坚韧、深谋远虑、外冷内热
- **动力**：家族复仇→探索真相→保护一切

### 境界体系

1. 凡躯境（觉醒者）
2. 星元境（行星级）
3. 恒耀境（恒星级）
4. 星域境（域主级）
5. 界主境（界主级）
6. 不朽境（不朽者）
7. 造化境（造化者）
8. 混元境（混元主）
9. 道祖境（道祖）
10. 超脱境（至高者）

### 主要势力

- 苍穹帝国（人族，修科并重）
- 机械联盟（AI主导，科技至上）
- 修真联盟（传统修真）
- 虚空商会（商业中立）
- 幽冥府（下界，掌管轮回）

## 使用指南

### 开始写作前

1. 阅读 `.cursorrules` 了解写作规范
2. 查阅 `01-worldbuilding/power-system.md` 熟悉力量体系
3. 查阅 `02-characters/protagonist.md` 熟悉主角设定
4. 查阅 `03-outline/volume-01-outline.md` 了解当前章节大纲

### 写作过程中

1. 使用 `@` 引用相关设定文档保持一致性
2. 参考 `05-materials/` 下的素材库
3. 使用 `06-templates/prompts-library.md` 中的提示词
4. 每章完成后更新 `03-outline/plot-threads.md`

### AI辅助写作

使用 `06-templates/prompts-library.md` 中的提示词模板，可以让AI辅助：
- 生成战斗场景
- 扩写修炼描写
- 润色对话
- 检查一致性
