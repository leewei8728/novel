## 二、常用公式

### 战力计算公式

```
综合战力 = 基础战力 × (1 + 功法加成 + 武器加成 + 金手指 + 状态)
```

**示例**(陆辰凡躯6阶):
```
5,000 × (1 + 0.8 + 0.2 + 0.4 + 0.1) = 13,000
```

---

### 境界压制公式

```
实际战力 = 基础战力 ÷ 压制系数
```

**压制系数**:
- 高1小阶: ÷1.5
- 高1大境界: ÷10

---

### 突破时间估算

```
突破时间 = 标准时间 ÷ (天赋系数 × 资源系数)
```

**主角系数**: ×3-5(混沌体+资源)

## 八、应急工具

### 卡文时的应急措施

1. **跳过这章**: 先写后面的
2. **写日常**: 人物互动章,不推进主线
3. **写回忆**: 补充背景故事
4. **查大纲**: 看看后面的精彩,找回动力
5. **休息**: 不要硬写

---

### 写作状态不佳时

**降低难度**:
- 不写战斗,写日常
- 不写新场景,写熟悉的
- 不写复杂情节,写过渡

**保持手感**:
- 哪怕只写500字,也要写
- 保持写作习惯
