# 正式章节目录

本目录用于存放小说的正式章节内容。

## 目录结构

```
04-chapters/
├── volume-01/          # 第一卷：觉醒
│   ├── chapter-001.md  # 第1章
│   ├── chapter-002.md  # 第2章
│   └── ...
├── volume-02/          # 第二卷：学院
│   └── ...
└── ...
```

## 章节命名规则

- 文件名格式：`chapter-XXX.md`
- XXX为三位数字章节号，如001、002、100

## 写作流程

1. 查阅 `03-outline/volume-XX-outline.md` 了解本章大纲
2. 使用 `06-templates/chapter-template.md` 作为章节模板
3. 写作过程中参考 `.cursorrules` 保持一致性
4. 完成后更新 `03-outline/plot-threads.md` 记录伏笔
