### 1.2 换算规则

- **基准**: 1下品灵石 = 100信用点
- **倍率**: 每提升一档 = ×100
- **逆向兑换**: 高等货币→低等货币(1:100),但需要5%手续费
- **正向兑换**: 低等货币→高等货币(100:1),需要10%手续费

## 六、经济系统特殊规则

### 6.1 灵石流通规则

1. **修士优先**: 凡人持有大量灵石会被怀疑来源
2. **实名交易**: 大额交易(>100中品灵石)需要登记
3. **税收**: 正规交易抽成5-10%
4. **黑市**: 匿名交易但风险高

### 6.2 通货膨胀

- **稳定期**: 信用点贬值<1%/年
- **灵石**: 相对保值,甚至升值
- **高境界货币**: 越高越保值

### 6.3 跨星域汇率

- **核心星域**: 1:1标准汇率
- **边境星域**: 0.8-0.9:1(略贬值)
- **荒僻星球**: 0.5-0.7:1(大幅贬值)

**青云星**: 属边境星域,约0.85:1主星汇率

## 八、使用说明

### 8.1 写作时参考原则

1. **境界匹配**: 修士的财富应与境界相符
2. **收入合理**: 收入来源要有逻辑
3. **消耗真实**: 修炼、生活、战斗都需要成本
4. **通货一致**: 同一物品价格前后要对应

### 8.2 价格锚定

- **1下品灵石 = 100信用点** (基准)
- **一顿营养膏 = 10信用点** (最低生活)
- **一个月贫民生活 = 1,000信用点** (温饱线)
- **凡躯1阶日常修炼 = 5下品灵石/日** (修炼成本)

### 8.3 后续卷调整

- 第2卷起,主角进入学院,经济来源转为任务奖励
- 第3卷起,主角境界提升,货币单位升至中品、上品灵石
- 第5卷起,开始涉及极品灵石和仙晶
