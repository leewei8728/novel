## 三、时间线特殊说明

### 3.1 叙事时间与实际时间

**前期(第1-5卷)**:
- 叙事时间 = 实际时间
- 详细描写每个月甚至每天
- 读者跟随主角成长

**中期(第6-8卷)**:
- 叙事时间 < 实际时间  
- 重点事件详写,日常略写
- 时间跳跃增多

**后期(第9-10卷)**:
- 叙事时间 << 实际时间
- 只写关键节点
- 数十年甚至百年一笔带过

### 3.2 时间跳跃处理

**短期跳跃**(数天到数月):
```
"三个月后,陆辰从闭关中醒来..."
```

**中期跳跃**(数月到数年):
```
"时光荏苒,转眼三年过去。
这三年里,陆辰游历了十数个星域..."
```

**长期跳跃**(数十年到百年):
```
"五十年,对界主境强者来说不过弹指一挥间。
这五十年中,宇宙局势风云变幻..."
```
