# 世界观渐进展开计划

## 说明

本文档规划世界观信息的渐进式披露,确保"随着剧情有计划的慢慢张开",避免信息过载,保持神秘感。

---

## 一、信息披露总原则

### 1.1 展开策略

```
世界观展开 = 主角视角 + 境界对应 + 剧情需要
```

**原则**:
- 主角知道什么,读者知道什么
- 境界到了,世界观展开到哪
- 剧情需要时,适度超前透露
- 保留神秘感,不要一次说完

### 1.2 展开速度

| 卷数 | 展开比例 | 累计 | 说明 |
|-----|---------|------|------|
| 第1卷 | 5% | 5% | 青云星,修炼基础 |
| 第2卷 | 10% | 15% | 苍穹星域,学院,虚拟宇宙 |
| 第3卷 | 15% | 30% | 多星域,其他文明 |
| 第4卷 | 15% | 45% | 星域级格局,上界传说 |
| 第5卷 | 15% | 60% | 上界初步,身世真相 |
| 第6-7卷 | 20% | 80% | 上界详细,法则体系 |
| 第8卷 | 10% | 90% | 宇宙真相,历史 |
| 第9-10卷 | 10% | 100% | 终极秘密,多元宇宙 |

## 七、描写技巧:从微观到宏观

### 7.1 视角变化

**第1卷**(微观):
```
陆辰站在贫民区的街角,看着前方拥挤的人群。
这就是他的世界,灰扑扑的,狭小的,压抑的。
```

**第3卷**(中观):
```
星舰穿过星云,陆辰透过舷窗看到数十个星球在远方闪烁。
每一个星球,都可能有无数生命,无数故事。
而这,只是苍穹星域的一角。
```

**第8卷**(宏观):
```
陆辰站在原初星域的虚空中,看着整个宇宙。
无数星域,如同棋盘上的棋子,在法则的编织下运转。
而在这棋盘之外,还有更广阔的世界。
```

### 7.2 信息密度控制

**第1卷**: 
- 信息密度低
- 详细描写单一场景
- 读者跟随主角慢慢理解

**第5卷**:
- 信息密度中等
- 大量新设定涌入
- 但主角境界够,可以理解

**第9卷**:
- 信息密度高
- 终极真相揭示
- 境界够高,可承载

## 八、世界观描写示例

### 8.1 第1卷:微观世界

```

```

```
陆辰在青龙星域的一座修真星球上,
看到无数修士御剑飞行,宗门林立。

但这些,不过是九大星域之三。
这个宇宙,还有太多他未曾见过的风景。
```

**特点**: 宇宙级视野

```
界主境的陆辰,站在宇宙的尺度思考。
九大星域,不过是中界的冰山一角。
上界九重天,下界六道,三界构成这个宇宙。
```

- 提及: "虚拟宇宙是宇宙级网络"

- 详细: 结构、功能、使用方式

## 十一、使用指南

### 11.1 写作时的自检

在引入新设定前问自己:
1. ✅ 主角境界到了吗?
2. ✅ 剧情需要吗?
3. ✅ 会不会信息过载?
4. ✅ 保留神秘感了吗?

### 11.2 避免的错误

❌ **过早透露**: 第1卷就讲不朽境
❌ **信息倾倒**: 一章讲完所有世界观
❌ **前后矛盾**: 前面说没有上界,后面突然有
❌ **缺乏铺垫**: 突然出现从未提及的设定
