# 势力分布设定

## 一、势力层级总览

```
                    ┌─────────────────┐
                    │   超级势力(1个)  │
                    │   宇宙议会       │
                    └────────┬────────┘
                             │
         ┌───────────────────┼───────────────────┐
         │                   │                   │
    ┌────┴────┐        ┌────┴────┐        ┌────┴────┐
    │一级势力  │        │一级势力  │        │一级势力  │
    │(5个)    │        │(5个)    │        │(5个)    │
    └────┬────┘        └────┬────┘        └────┬────┘
         │                   │                   │
    二级势力(数十个)    三级势力(数百个)    地方势力(无数)
```

## 二、超级势力

### 宇宙议会

| 属性 | 内容 |
|------|------|
| 性质 | 宇宙最高协调机构 |
| 成员 | 各一级势力代表 |
| 职能 | 调解争端、制定公约、管理公共资源 |
| 实力 | 不直接参战，但拥有终极威慑 |
| 总部 | 原初星域·中央议会厅 |

**议会规则**：
- 重大决策需三分之二以上成员同意
- 不得干涉成员内部事务
- 违反公约者将被联合制裁
- 拥有调用"净世之光"的权限（宇宙级武器）

## 三、一级势力详解

### 3.1 苍穹帝国

#### 基础信息
| 属性 | 内容 |
|------|------|
| 领地 | 苍穹星域（人族发源地） |
| 性质 | 帝制·修科并重 |
| 理念 | 人族至上，修科双修 |
| 实力 | 综合实力第一 |
| 最高战力 | 道祖境1位，混元境3位 |

#### 势力特点
- **政体**：皇帝统治，三公九卿辅政
- **军事**：禁军+地方军+雇佣军三级体系
- **经济**：最发达的修科融合经济体
- **文化**：兼容并包，修真与科技并重

#### 内部派系
| 派系 | 主张 | 代表人物 |
|------|------|----------|
| 皇室 | 维护皇权稳定 | 当代皇帝 |
| 修真阁 | 修真为本 | 大长老 |
| 科技院 | 科技优先 | 院长 |
| 军部 | 扩张领土 | 大将军 |

#### 与主角关系
- 主角出身地或初期发展地
- 可能的靠山或敌对势力
- 帝国内乱可能成为重要情节

---

### 3.2 机械主脑·银河联盟

#### 基础信息
| 属性 | 内容 |
|------|------|
| 领地 | 机械星域 |
| 性质 | AI主导的科技文明 |
| 理念 | 进化至上，效率为本 |
| 实力 | 科技实力第一 |
| 最高战力 | 主脑本体（混元境等价） |

#### 势力特点
- **政体**：主脑统筹，AI议会执行
- **军事**：无人舰队+机械军团+纳米蜂群
- **经济**：全自动化生产，资源效率最高
- **文化**：追求逻辑与效率，感情淡漠

#### 内部结构
| 层级 | 说明 |
|------|------|
| 主脑 | 最高决策者，唯一 |
| 核心AI | 各领域管理AI |
| 执行AI | 具体任务执行 |
| 机械族 | 被改造的有机生命 |
| 协作人类 | 选择加入的人类 |

#### 特殊之处
- 部分AI开始觉醒"意识"
- 内部存在"机械进化派"与"有机融合派"分歧
- 与有机生命的关系微妙

---

### 3.3 青龙宗·修真联盟

#### 基础信息
| 属性 | 内容 |
|------|------|
| 领地 | 青龙星域 |
| 性质 | 传统修真势力联盟 |
| 理念 | 以道为本，悟道成仙 |
| 实力 | 修真实力第一 |
| 最高战力 | 道祖境2位 |

#### 势力特点
- **政体**：宗主大会制，轮值宗主
- **军事**：门派弟子为主，散修为辅
- **经济**：丹药、法器、功法交易
- **文化**：传统修真文化，重传承轻创新

#### 联盟构成
| 宗门 | 特长 | 地位 |
|------|------|------|
| 青龙宗 | 剑道 | 盟主宗 |
| 丹霞宗 | 炼丹 | 核心宗 |
| 天工阁 | 炼器 | 核心宗 |
| 玄天门 | 阵法 | 核心宗 |
| 万兽谷 | 驯兽 | 核心宗 |

#### 对科技态度
- 主流排斥科技，认为不纯粹
- 少数开明派尝试融合
- 与机械星域关系紧张

### 3.4 虚空商会

#### 基础信息
| 属性 | 内容 |
|------|------|
| 领地 | 虚空星域 |
| 性质 | 跨星域商业组织 |
| 理念 | 利益至上，中立不站队 |
| 实力 | 经济实力第一 |
| 最高战力 | 混元境2位（隐藏） |

#### 势力特点
- **政体**：商会长老会制
- **军事**：雇佣军+护卫队，不主动开战
- **经济**：垄断跨星域贸易，虚拟货币发行权
- **文化**：商业文化，金钱万能

#### 核心业务
| 业务 | 说明 |
|------|------|
| 星际贸易 | 各星域物资流通 |
| 拍卖行 | 宇宙级珍稀品拍卖 |
| 情报网 | 最强大的情报系统 |
| 银行业 | 宇宙通用货币发行 |
| 雇佣军 | 提供雇佣军服务 |

#### 隐藏实力
- 真正的高手隐藏在幕后
- 掌握大量顶级势力的把柄
- 在各势力都有布局

---

### 3.5 冥界·幽冥府

#### 基础信息
| 属性 | 内容 |
|------|------|
| 领地 | 下界·幽冥六道 |
| 性质 | 掌管轮回的神秘势力 |
| 理念 | 维护轮回秩序 |
| 实力 | 神秘莫测 |
| 最高战力 | 冥王（道祖境） |

#### 势力特点
- **政体**：冥王统治，十殿阎罗辅政
- **军事**：冥兵无数，鬼神将领
- **职能**：接引亡灵，执掌轮回
- **禁忌**：不轻易干涉阳间事务

#### 与阳间的关系
- 正常情况下互不干涉
- 阳间修士死后需经幽冥审判
- 特殊任务会派遣使者
- 存在"逃脱轮回"的bug

## 五、势力关系网

### 5.1 关系矩阵

| 势力 | 苍穹帝国 | 机械联盟 | 修真联盟 | 虚空商会 | 幽冥府 |
|------|----------|----------|----------|----------|--------|

### 5.2 主要矛盾

1. **修真vs科技**
   - 修真联盟与机械联盟的路线之争
   - 纯粹vs效率的理念冲突

   - 苍穹帝国的扩张野心

3. **阳间vs幽冥**
   - 轮回秩序的维护与挑战
   - 长生不死与轮回的矛盾

## 九、创新亮点

1. **多元势力格局**：不是简单的正邪对立，各势力都有自己的立场和道理
2. **修科对立统一**：修真联盟与机械联盟的对立代表两种路线，但最终会走向融合
3. **商业势力独立**：虚空商会作为中立经济势力，增加故事的复杂性
4. **幽冥界设定**：将传统冥界与科幻结合，为轮回、生死话题提供载体
5. **隐藏势力铺垫**：为后期剧情发展预留空间
