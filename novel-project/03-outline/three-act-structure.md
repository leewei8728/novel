# 三幕结构规划

## 总体架构

### 基础数据
| 属性 | 数值 |
|------|------|
| 总字数 | 200万字 |
| 总卷数 | 10卷 |
| 每卷字数 | 约20万字 |
| 每卷章节 | 约80章 |
| 每章字数 | 2500-3000字 |
| 总章节数 | 约800章 |

### 三幕比例
| 幕 | 字数 | 卷数 | 比例 |
|----|------|------|------|
| 第一幕（建立） | 0-40万字 | 第1-2卷 | 20% |
| 第二幕（对抗） | 40-160万字 | 第3-8卷 | 60% |
| 第三幕（解决） | 160-200万字 | 第9-10卷 | 20% |

---

## 第一幕：建立（0-40万字）

### 幕的目标
- 介绍主角和世界观
- 建立初期人际关系
- 完成第一阶段目标（家族复仇）
- 为后续格局扩展铺垫

### 第1卷：觉醒（0-20万字）

#### 卷信息
| 属性 | 内容 |
|------|------|
| 主题 | 觉醒与崛起 |
| 主角境界 | 凡人→凡躯境9阶 |
| 主要场景 | 青云星 |
| 时间跨度 | 宇宙历10000年（约1年） |

#### 主线情节
1. **开篇（1-10章）**
   - 主角困顿处境介绍
   - 因果之瞳觉醒
   - 第一次使用能力脱险

2. **初步崛起（11-30章）**
   - 发现混沌体特性
   - 找到修炼方法
   - 与林若琳相识
   - 初步报复赵家

3. **势力纷争（31-50章）**
   - 卷入青云星势力争斗
   - 遇到玄明老人
   - 获得传承/资源
   - 实力快速提升

4. **第一次高潮（51-70章）**
   - 赵家反扑
   - 生死危机
   - 突破关键境界
   - 击败赵天霸

5. **收尾与铺垫（71-80章）**
   - 发现幕后还有黑手
   - 决定离开青云星
   - 前往苍穹学院
   - 伏笔：母亲的线索

#### 支线情节
- 林若琳的药师成长
- 妹妹陆雪的修炼启蒙
- 玄明老人的身份暗示

#### 本卷高潮
第60章左右：与赵天霸的决战，以弱胜强，家族复仇第一阶段完成

#### 本卷钩子
赵天霸临死前透露："你以为这就完了？陆家得罪的人，你永远想不到有多可怕！"

## 创作备注

### 节奏把控原则
1. 每卷都要有明确的目标和高潮
2. 升级节奏前快后慢，符合修真小说规律
3. 伏笔要提前埋设，揭示要有节奏
4. 支线不能喧宾夺主，要适时收束

### 注意事项
1. 第一幕是吸引读者的关键，节奏要快
2. 第二幕是核心内容，要丰富但不拖沓
3. 第三幕要收束有力，不能烂尾
4. 每卷结尾都要留钩子
